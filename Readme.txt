Zen Cart v1.5.8 Japanese language pack

All information for this plugin are in doc folder in index.html file. Please have a look there.

このプラグインのすべての情報は、index.html ファイルの doc フォルダーにあります。 ぜひそこをご覧ください。

This language pack includes not only Japanese language files but although core files and database modifications adding possibility to use furigana and  products dimension anywhere in the cart system.
This allows use of Japanese Yamato express delivery, Nekoposu, Sagawa express delivery, Japan Post Yupack, LetterPack Lite, LetterPack Plus, International ePackets and EMS services. All these shipping modules files are included. Some payment and order total modules that complete shipping modules (c.o.d.) are although provided.

この日本語パックには、言語ファイルだけでなく、コア ファイルとデータベースの変更が含まれており、ふりがな、商品寸法、バーコードを使用する可能性が追加されています。
これにより、日本のヤマト便、ネコポス、佐川急便、日本郵便ゆうパック、レターパック ライト、レターパック プラス、および EMS サービスを利用できます。配送、支払い、および注文の合計モジュール ファイルが含まれています。
